# LangChain

LangChain is a framework for building applications powered by LLMs.

## 1. Components

### 1. LLMs

- OpenAI, Claude, Gemini, local models, etc.
- Unified interface, easy to switch between models

### 2. Prompt Templates

- Parameterize and structure prompts

### 3. Chains

- Connect multiple steps into a workflow  
- Example: Summarize → Translate → Format output

### 4. Memory

- Short-term / long-term context management

### 5. Tools

- Enable models to query databases, search the web, and execute code

## 2. Applications

### Common Use Cases

1. RAG (Retrieval-Augmented Generation): document Q&A / knowledge base assistants  
2. AI assistants / copilots  
3. Multi-step reasoning systems  
4. Automated task execution
