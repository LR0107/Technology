# Node.js (Windows)

## 1. Install Node.js

Download and install Node.js:  
https://nodejs.org

CMDï¼šVerify installation
`node -v`
`npm -v`

## 2. Create a Vue Project

Create and enter the working directory:

`cd D:\projects`

Create a Vue project with Vite (files are generated automatically):

`npm create vite@latest vue-demo`

## 3. Write Code

Write your code in the `src` directory and its subdirectory `components`.

## 4. Run the Project

Start the development server:

`npm run dev`

Copy the displayed URL and open it in a browser.

Stop the server:

`Ctrl + C`
