# Windows Remote Connection to Linux

## Linux

**Install OpenSSH Server**

sudo apt install openssh-server  



**Start SSH**

sudo systemctl start sshd  



**Check SSH Status**

sudo systemctl status sshd  



**View IP Address**

ifconfig  //Requires net-tools



## Windows

1. **Ping the IP**

2. **Connect via SSH**
   
   ssh username@server_ip
