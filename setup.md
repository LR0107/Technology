**Create virtual environment**
`python -m venv venvname`   

**Activate**
`venv\Scripts\activate`  
Success example: `(venv) C:\Users\`  

**Install dependencies** (requirements file must be in the same directory) 
`pip install -r requirements.txt`

**Export environment**  
`pip freeze > requirements.txt`    

**Deactivate**
`deactivate`
